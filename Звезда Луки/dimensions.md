# Звезда Луки — Описание конструкции и пропорции

## Общие габариты (базовая позиция — крест)

| Параметр | Размер |
|---|---|
| Размах от конца до конца (через центр) | **17.5 см** |
| Толщина (объект лежит плоско) | **~1.5–2 см** |
| Центральный кластер (кольцо + блоки) | **~4.5 см × 4.5 см** |
| Длина одного рычага от шарнира до конца | **~3.5 см** |
| Расстояние от края центра до конца рычага | **~6.5 см** |

---

## Состав конструкции: 16 деталей

### 1. Бирюзовые блоки (× 4)
- Расположение: центральный кластер 2×2
- Функция: несущий узел; принимает шар сверху и соединяется с Г-коннектором сбоку
- Сверху блока — гнездо для шара (отдельная деталь)
- Цвет: бирюзовый / тёмно-зелёный

### 2. Шары — дисковидные шарниры (× 4)
- Форма: дисковидная / полусферическая (не полный шар — чашеобразная форма)
- Расположение: по одному сверху на каждом бирюзовом блоке
- Крепление: вставляются штырём в верхнее отверстие бирюзового блока (штырь скрыт внутри, снаружи не виден)
- Функция: шаровой шарнир — рычаг касается шара своей внешней (вогнутой) стороной, фиксируясь на нём без полного охвата; даёт свободу движения во всех направлениях
- Цвет: чёрный

### 3. Чёрные Г-коннекторы (× 4)
- Форма: дугообразная (плавная кривая, не угловая)
- Функция: соединяют соседние бирюзовые блоки — каждый коннектор = одна дуга между двумя соседними блоками
- 4 дуги образуют замкнутое кольцо вокруг центра
- Внутри кольца — пустой центр (каркасная конструкция)
- Цвет: чёрный

### 4. Рычаги (× 4) — 2 белых + 2 чёрных
- Все 4 одинаковой формы, различаются только цветом
- Крепление: через шар-шарнир, сбоку бирюзового блока
- Форма: асимметричный Г (одно плечо длиннее другого)
  - Короткое плечо → идёт наружу, это «рука/нога» объекта (~3.5 см)
  - Длинное плечо → внешней (вогнутой) стороной касается шара-шарнира
- Угол между плечами Г: приблизительно 90°
- Расположение в базовой позиции:
  - **Белые** (2 шт.) — горизонтально, влево и вправо
  - **Чёрные** (2 шт.) — вертикально, вверх и вниз

## Пропорциональная схема (вид сверху, базовая позиция)

Обозначения:
- `[Б]` — бирюзовый блок
- `●` — чёрный шар-шарнир (сверху блока)
- `╗ ╔ ╝ ╚` — чёрный Г-коннектор (дуга между блоками)
- `════` — длинное плечо рычага (к шару)
- `│` — короткое плечо рычага (наружу, «рука/нога»)

```
                    │  ~3.5 см
                    │ (чёрный рычаг ↑)
                    │
              ╔═════╪═════╗
              ║  [Б]●  [Б]║
 ~3.5 см ────╪──●      ●──╪──── ~3.5 см
(белый ←)    ║  [Б]●  [Б]║    (→ белый)
              ╚═════╪═════╝
                    │
                    │ (чёрный рычаг ↓)
                    │  ~3.5 см

←————————————— 17.5 см ————————————————→
         центр ~4.5 см × 4.5 см
```

**Вид сбоку (упрощённо):**
```
   ════[Б]════   ← длинное плечо рычага лежит вдоль блока
       ●          ← шар сверху блока
       │           ← короткое плечо рычага уходит наружу (перпендикулярно)
```

## Пропорции (относительные, от общего размаха)

| Часть | Размер | % от размаха |
|---|---|---|
| Центральный кластер | ~4.5 см | ~26% |
| Один рычаг (от шарнира до конца) | ~3.5 см | ~20% |
| Один «рукав» от центра до конца | ~8.75 см | ~50% |

Соотношение центр : рукав ≈ **1 : 2**

---

## Механика движения

**Два независимых параметра:**
1. **Поворот рычага** — рычаг вращается в шарнире бирюзового блока (360°, полный оборот)
2. **Скручивание секции** — бирюзовый блок + его рычаг могут разворачиваться относительно соседних блоков через Г-коннектор

Детали **никогда не снимаются** — объект всегда цельный.

---

## Цветовая схема

| Цвет | Детали |
|---|---|
| Бирюзовый | 4 центральных блока |
| Чёрный | 4 шара-шарнира + 4 Г-коннектора + 2 вертикальных рычага |
| Белый / кремовый | 2 горизонтальных рычага |

Контраст белого и чёрного на рычагах создаёт визуальную ориентацию — сразу видно, куда «смотрит» объект.

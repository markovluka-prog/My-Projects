# Контекст для Claude — Звезда Луки

## Статус проекта
Прототип готов (физически). Патент не оформлен.
3D модель в Blender — в процессе.

## Что делали в последний раз
- Создали blender_model.py — скрипт для Blender (запуск: blender --background --python blender_model.py)
- Сделали несколько итераций рендера, добиваясь похожести на фото
- Финальная структура модели (актуальная):
  - Центр: 2 бирюзовых блока 2×2, стопкой по Y (итого 2×4), со студами и отверстиями
  - Чёрные щиты N/S: крепятся напрямую к торцам центра (без дуг)
  - Белые щиты E/W: соединяются через 4 J-образных дуги (по 2 на каждый щит, огибают углы NE/SE и NW/SW)
  - Вид строго сверху, ортографическая камера
  - Рендер: Cycles 128 spp, denoising
  - Файлы: blender_model.py, zvezda_luki.blend, zvezda_luki_render.png

## Важные детали
- Базовая форма — плоский крест (+), НЕ X-форма
- Трансформация через ДВА параметра: поворот Г-рычагов + опциональное скручивание секций
- Детали никогда не снимаются — объект всегда цельный
- Запуск модели: cd "Звезда Луки" && blender --background --python blender_model.py

## Что нельзя трогать / менять
- Принцип цельности — детали не переставляются
- Базовая форма (крест) — официальная стартовая позиция

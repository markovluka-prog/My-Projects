## Что это

Простой GPT-подобный чат-бот, написанный с нуля на Python без ML-библиотек.
Работает на основе словаря слов и алгоритма схожести по буквам.

## Архитектура

| Файл            | Роль                                                  |
|-----------------|-------------------------------------------------------|
| GPT.py          | UI-оболочка (Pythonista ui + WKWebView)               |
| INIT.py         | Ядро: `learn(text)` и `answer(text)`                  |
| Functions.py    | Утилиты: очистка текста, работа со словарями, веса    |
| Root_model.py   | Алгоритм схожести двух слов по позициям букв          |
| Base_weight.py  | Ранжирование слов по частоте (bubble-sort-like)       |
| Dictionary.py   | Хранение словаря слов (Dictionary.pkl)                |

## Как работает

1. `learn(text)` — добавляет новые слова из текста в словарь
2. `answer(text)` — для каждого слова вопроса считает вес и схожесть со всеми словами словаря, возвращает самое близкое слово
3. Вес слова растёт каждый раз, когда оно используется в запросе

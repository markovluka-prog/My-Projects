# Контекст проекта: LIT GPT

## Статус
Первый рабочий код. Запускается в Pythonista на iOS. Базовый функционал работает.

## Что было сделано (последняя сессия — 2026-03-12)

### Исправлены баги:
1. **`Functions.py` → `add_two_dicts`**
   - Было: `p = 0` → score из `first_dict` игнорировался, накопление не работало
   - Стало: `p = first_dict[a] + second_dict[a]` — теперь score честно суммируется

2. **`Root_model.py` → `find_the_same_letters`**
   - Было: `.index()` всегда находил первое вхождение буквы → дубли не учитывались
   - Стало: `enumerate` + `used_big` — каждая позиция засчитывается только раз

## Архитектура (коротко)
- `learn(text)` → добавляет слова в `Dictionary.pkl`
- `answer(text)` → для каждого слова вопроса считает вес (`Base_weight`) + схожесть (`Root_model`) со всеми словами словаря → возвращает топ-1 слово

## Что можно улучшить дальше
- Отвечает только 1 словом — добавить ответы из нескольких слов
- Нет контекста между вопросами — добавить историю диалога
- `learn` не обновляет веса слов — только добавляет в словарь
- Перенести с Pythonista на обычный Python + HTML-интерфейс

## Важно не ломать
- `Dictionary.pkl` и `GPT.pkl` — файлы данных, не трогать логику сохранения без причины
- `ui.load_view("GPT")` — GPT.pyui содержит разметку интерфейса для Pythonista

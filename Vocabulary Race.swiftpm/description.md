# Vocabulary Race

## Что это
Настольная словесная игра с элементами бродилки для 2–4 игроков на одном устройстве. Игроки составляют слова из букв выпавшей карточки, зарабатывают очки и двигают фишки по замкнутому маршруту из 48 клеток. Первый до 50-й клетки — побеждает.

## Как работает

### Экраны
1. **Экран загрузки** — логотип, фоновая загрузка словаря
2. **Диалог** — выбор количества игроков (2/3/4)
3. **Главное поле** — маршрут с фишками, текущий ход, очки
4. **Экран-заглушка** — "Синий игрок, нажмите чтобы начать" (защита от подглядывания)
5. **Экран задания** — буквы карточки, поле ввода слова, таймер 60 сек
6. **Экран победы** — затемнение поля + "[ЦВЕТ] ВЫИГРАЛ"

### Структура хода
1. Тянется карточка из колоды
2. Если спецкарточка — сохраняется у игрока, тянется следующая
3. Если у игрока есть спецкарточки — выбор: использовать или играть обычно
4. **Гонщик** → +N очков, ход заканчивается
5. **Судья** → выбрать жертву, +N себе, -N жертве
6. **Обычная игра** → экран задания, 60 сек, составить N слов
7. Успех → +N очков, фишка вперёд; провал → -N очков, фишка назад
8. Ход переходит к следующему игроку

*N = количество слов (сейчас = 2, в будущем — бросок кубика 1–6)*

## Игровое поле
- 48 клеток, замкнутый маршрут, движение по часовой стрелке
- Старт — правый верх; вниз → влево → вверх → к старту
- Фишки анимировано двигаются по одной клетке, задержка 0.4 сек
- Две фишки на одной клетке — одна смещается вниз, не "бьют" друг друга
- Поле: `assets/field.png`, маршрут — 48 координат вручную

## Колода карточек

### Обычные карточки
- Русские слова длиной 7–21 букву из `dictionary.txt`
- Уже использованные не повторяются до полного исчерпания (затем сброс)
- Одно и то же слово не выпадает дважды подряд

### Специальные карточки (4 штуки в колоде)
| Карточка | Кол-во | Действие |
|----------|--------|----------|
| 🏎️ Гонщик | 2 | Автоматически продвинуться вперёд без задания |
| ⚖️ Судья | 2 | Забрать очки у выбранного соперника |

- Накапливаются у игрока
- Перед ходом можно выбрать — использовать или нет
- После использования уходят из игры навсегда

## Экран задания

**Что видит игрок:**
- Буквы исходного слова — каждая отдельная кнопка
- Поле набираемого слова — над кнопками
- Кнопка OK, кнопка ❌ (сброс)
- Счётчик слов: `1/2`
- Таймер: обратный отсчёт `60⏰`
- Список использованных слов (справа)

**Правила:**
- Буквы нажимаются по одной, нажатая исчезает с поля
- Минимум 2 буквы
- Слово должно быть в словаре
- Нельзя использовать исходное слово
- Нельзя повторять уже составленные в этом ходу слова
- ❌ возвращает все буквы обратно

## Система очков
- Очки = номер клетки фишки (score = position)
- Успех: +N, фишка вперёд
- Провал: -N, фишка назад (ниже 0 — остаётся на месте)
- Победа: достичь клетки 50

## Словарь
- Файл: `dictionary.txt`
- Слова построчно, длина 7–21 символ
- Проверка: точное совпадение со строкой из словаря

## Незавершённые механики
| Что | Статус |
|-----|--------|
| Кубик 1–6 для N | Захардкожено в 2 |
| Карточка судьи — потоки | Баг: неправильный threading |
| Движение назад при штрафе | Частично реализовано, есть баги |
| Конец игры — перезапуск | Не реализован, игра замирает |
| Мультиплеер на разных устройствах | Не планируется |

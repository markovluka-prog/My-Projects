## Ключевые договорённости
- **Формула 1 (давление гравитации):** `F = (k / k_макс) · b · (r / k_макс)³` если k ≤ k_макс, иначе F = 0. b = масса большего объекта (кг), r = радиус меньшего объекта (м). Линейная часть × объёмная часть. Нарастает с расстоянием.
- **Формула 2 (поле):** `k_макс = r²/2` — каждый слой на радиусе ρ даёт поле ρ, сумма всех слоёв = r²/2. Земля: 136 АЕ, Солнце: 1.6 млн АЕ.
- Любой объект = своё ядро
- Антигравитон = электрон (e-), условие: отрицательных ионов в 2× больше положительных
- Не сравнивать формулу 1 с реальным ускорением свободного падения — разные величины

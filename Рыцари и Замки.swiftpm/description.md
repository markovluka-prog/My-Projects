# Рыцари и Замки

## Что это
Пошаговая стратегия на сетке с тёмным средневековым сеттингом. Игрок расставляет и двигает отряды по клеткам, атакует врага, управляет позицией и ресурсами. Есть сюжетная кампания и свободный режим против ИИ.

## Как работает
- При запуске открывается главный экран (меню)
- Два режима: сюжетная кампания и свободная битва
- Игровое поле — canvas 480×960, слева сетка, справа sidebar с информацией о юнитах, ходе и действиях
- Игрок и ИИ ходят по очереди
- В кампании бои идут по главам с нарастающей сложностью ИИ

## Функции
- [ ] Пошаговые бои на сетке
- [ ] 7 уровней ИИ: Новичок, Ученик, Боец, Воин, Тактик, Мастер, Легенда
- [ ] Сюжетная кампания: 3 главы (Первые шаги, Морской ветер, Разлом)
- [ ] Именованные персонажи (Ричард, Соколин, Бен)
- [ ] Разные типы юнитов (меч, лук, посох, конница и др.)
- [ ] Боевая музыка и фоновая музыка зала
- [ ] Горячее обновление игры через папку Documents (GitHubUpdater)

## Дизайн
- Цветовая схема: тёмная средневековая — фон `#1a1410`, панели `#221a14`, текст `#d9ccb5`, акцент золото `#ffe3b2`
- Шрифты: Segoe UI / Tahoma (системные)
- Общее настроение: мрачное, тактическое, пиксельное

## Технологии
- Swift Playgrounds + WKWebView — обёртка
- HTML / CSS / JS — весь интерфейс и логика
- Canvas API — рендер игрового поля
- GitHubUpdater (Swift) — подгрузка обновлений из Documents

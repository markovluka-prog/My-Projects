# ✅ Чеклист настройки

## 1️⃣ Настройте GitHubUpdater.swift

Откройте `GitHubUpdater.swift` и измените:

```swift
private let repository = "ВАШ_USERNAME/ВАШ_REPO"  // Например: "ivan/knights"
private let folderName = "KnightsGame"             // Папка в репозитории
```

## 2️⃣ Проверьте структуру GitHub репозитория

Ваш репозиторий должен содержать:

```
https://github.com/ВАШ_USERNAME/ВАШ_REPO/tree/main
└── KnightsGame/
    ├── index.html           ← ОБЯЗАТЕЛЬНО
    ├── (другие файлы)
    ├── (подпапки)
    └── (ресурсы)
```

## 3️⃣ Проверьте, что репозиторий публичный

- Зайдите в Settings репозитория на GitHub
- Убедитесь что он Public (не Private)

## 4️⃣ Протестируйте приложение

### С интернетом:
1. Запустите приложение
2. Проверьте консоль Xcode
3. Должны увидеть: `🔄 Начинаем фоновое обновление...`
4. Приложение откроется мгновенно с Bundle версией
5. Через несколько секунд обновление скачается
6. Закройте и снова откройте приложение
7. В консоли: `📂 Загружаем из Documents (обновлённая версия)`

### Без интернета:
1. Включите режим полёта на устройстве
2. Запустите приложение
3. В консоли: `📡 Нет подключения к интернету, обновление пропущено`
4. В консоли: `📦 Загружаем из Bundle (версия по умолчанию)`
5. Игра работает нормально

## 5️⃣ (Опционально) Добавьте Debug UI

Если хотите видеть статус обновлений, откройте `MyApp.swift` и измените:

```swift
WindowGroup {
    ZStack {
        ContentView(updater: updater)
            .ignoresSafeArea()
        
        // Debug overlay — удалите перед релизом!
        VStack {
            Spacer()
            HStack {
                UpdateDebugView(updater: updater)
                Spacer()
            }
        }
        .padding()
    }
    .task {
        updater.checkForUpdatesInBackground()
    }
}
```

## 🐛 Отладка

### Очистить кэш и вернуться к Bundle версии:

Добавьте в ваш UI кнопку:

```swift
Button("Очистить кэш") {
    Task {
        await updater.clearCache()
    }
}
```

Или сбросьте через Device → Erase All Content and Settings в симуляторе.
